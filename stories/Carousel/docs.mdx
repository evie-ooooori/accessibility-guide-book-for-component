# Carousel

## 키보드 컨트롤

### 1. Shift + Tab`

- `Shift + Tab` 키를 통해 캐러셀 내부 초점 이동이 가능한 이전 요소로 초점을 이동시킵니다.

### 2. Tab`

- `Tab` 키를 통해 캐러셀 내부 초점 이동이 가능한 이전 요소로 초점을 이동시킵니다.

<br />

### 초점이 일시정지/재생 버튼 요소에 있는 경우

### 1. `Space` 또는 `Enter`

- 캐러셀의 일시정지/재생 상태를 전환합니다.

<br />

### 초점이 이전/다음 슬라이드 이동 버튼 요소에 있는 경우

### 1. `Space` 또는 `Enter`

- 슬라이드의 다음 또는 이전 슬라이드를 표시합니다.
