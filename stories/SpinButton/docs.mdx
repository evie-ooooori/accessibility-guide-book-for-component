# SpinButton

## 키보드 컨트롤

### 초점이 SpinButton에 있는 경우

### 1. `Arrow Down`

- `Arrow Down` 키를 통해 현재 값을 감소시킵니다.

### 2. `Arrow Up`

- `Arrow Up` 키를 통해 현재 값을 증가시킵니다.

### 3. `Home`

- `Home` 키를 통해 현재 값을 최소값으로 만듭니다.

### 4. `End`

- `End` 키를 통해 현재 값을 최대값으로 만듭니다.

### 5. `Page Down` (옵션)

- `Page Down` 키를 통해 현재 값을 `Arrow Down` 키를 통한 감소보다 큰 간격으로 값을 감소시킵니다.

### 6. `Page Up` (옵션)

- `Page Up` 키를 통해 현재 값을 `Arrow Up` 키를 통한 증가보다 큰 간격으로 값을 증가시킵니다.
