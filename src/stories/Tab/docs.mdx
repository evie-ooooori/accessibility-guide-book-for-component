# Tab

## 키보드 컨트롤

### 초점이 Tab 요소에 있는 경우

### 1. `Arrow Left`

- `Arrow Left` 키를 통해 이전 탭으로 초점을 이동시킵니다. (탭 버튼이 세로로 나열된 경우에는 `Arrow Up`)
  <br />
  첫번째 탭에서 `Arrow Left` 키를 누르면 마지막 탭으로 초점을 이동시켜 탭 리스트
  내에서 초점이 순환되도록 합니다.

### 2. `Arrow Right`

- `Arrow Right` 키를 통해 다음 탭으로 초점을 이동시킵니다. (탭 버튼이 세로로 나열된 경우에는 `Arrow Down`)
  <br />
  마지막 탭에서 `Arrow Right` 키를 누르면 첫번째 탭으로 초점을 이동시켜 탭 리스트
  내에서 초점이 순환되도록 합니다.

### 3. `Home`

- `Home` 키를 통해 첫번째 탭으로 초점을 이동시킵니다.

### 4. `End`

- `End` 키를 통해 마지막 탭으로 초점을 이동시킵니다.
