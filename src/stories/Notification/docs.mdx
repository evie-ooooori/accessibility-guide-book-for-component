# AlertDialog

## 키보드 컨트롤

### 초점이 AlertDialog 내부에 있는 경우

### 1. `Tab`

- `Tab` 키 동작을 통해 AlertDialog 내부 초점 이동이 가능한 다음 요소로 초점을 이동시킵니다.

### 2. `Shift + Tab`

- `Shift + Tab` 키 동작을 통해 AlertDialog 내부 초점 이동이 가능한 이전 요소로 초점을 이동시킵니다.

### 3. `Esc`

- `Esc` 키를 통해 AlertDialog를 닫습니다.
