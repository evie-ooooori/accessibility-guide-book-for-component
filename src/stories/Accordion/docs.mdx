# AccordionGroup

## 키보드 컨트롤

### 초점이 아코디언 헤더 요소에 있는 경우

### 1. `Space` 또는 `Enter`

- `Space` 또는 `Enter` 키를 통해 축소된 패널의 아코디언 헤더와 연관된 아코디언 패널을 확장합니다. 그리고 확장된 패널의 아코디언 헤더에 초점이 있을 때, 연관된 아코디언 패널을 축소합니다.

### 2. `Down Arrow`

- `Down Arrow` 키를 통해 다음 아코디언 헤더로 초점을 이동시킵니다.

### 3. `Up Arrow`

- `Up Arrow` 키를 통해 이전 아코디언 헤더로 초점을 이동시킵니다.

### 4. `Home`

- `Home` 키를 통해 첫 번째 아코디언 헤더로 초점을 이동시킵니다.

### 5. `End`

- `End` 키를 통해 마지막 아코디언 헤더로 초점을 이동시킵니다.
