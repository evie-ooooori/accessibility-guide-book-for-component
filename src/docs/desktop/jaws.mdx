# JAWS(Job Access With Speech)

JAWS는 Freedom Scientific에서 개발한 전 세계에서 가장 많이 사용하는 스크린 리더입니다. 유료로 가격은 라이선스 유형에 따라 다르며 가정용 라이센스의 경우 연간 95달러의 비용이 듭니다. JAWS는 유료이지만 가장 오래된 스크린 리더로 신뢰도 및 안정성에서 높게 평가 받고 있으며, 사용자의 선호도에 맞게 다양한 설정을 할 수 있어 인기가 많습니다. JAWS는 Chrome, Firefox 브라우저와의 호환성이 뛰어납니다.

```plaintext
💡 국내의 웹 접근성 품질인증마크 사용자 심사에서 사용되기 때문에 접근성 인증 마크 취득을 목적으로 한다면 기본적인 JAWS 사용법에 대해 이해하는 것이 좋습니다.
```

**[특징]**

- 웹 브라우저, 오피스 관련 프로그램(예: Microsoft Office, Google Docs 등) 등을 포함한 다양한 응용 프로그램 사용 가능
- 2개의 다국어 음성 합성 엔진(Eloquence, Vocalizer Expressive)을 통해 명확하고 이해하기 쉬운 오디오 출력 가능
- 다양한 언어의 음성 엔진을 무료로 다운로드하여 사용할 수 있도록 지원
- 마우스가 위치한 곳의 글자를 자동으로 읽어 쉽게 탐색 가능
- 각 단락의 첫 번째 줄이나 문장을 읽어 긴 문서를 빠르게 검토할 수 있는 훑어보기 기능 제공
- 고급 사용자 정의 옵션 제공으로 사용자에게 맞춤화된 경험 제공
- 이미지나 PDF 문서에 대한 OCR 기능
- 인쇄된 문서나 책에 직접 접근할 수 있는 PEARL 카메라 지원
- Windows 11, Windows 10, Windows Server 2019 및 Windows Server 2016 운영 체제에서 동작
- 오디오를 듣지 않고도 콘텐츠를 입력하거나 읽을 수 있는 새로 고침 가능한 점자 디스플레이와의 뛰어난 호환성
- 터치 명령을 사용하여 컨트롤 및 텍스트 읽기 가능

### JAWS 설치 방법

JAWS는 Freedon Scientific(https://support.freedomscientific.com)에서 다운로드 해야 합니다. `Downloads > JAWS`로 이동하여 해당 페이지(https://support.freedomscientific.com/Downloads/JAWS)에서 파일을 다운로드 후 JAWS를 설치합니다.

<img
  width="500"
  src="https://evie-ooooori.github.io/accessibility-guide-book-for-component/img/jaws-download.png"
/>

```plaintext
💡 JAWS는 유료 스크린 리더지만, 라이선스 키를 등록하지 않은 상태에서는 체험을 위해 40분 동안 무료로 사용할 수 있는 체험 모드로 실행됩니다. 40분이 지나면 Windows를 다시 시작해야만 체험 모드로 사용할 수 있습니다. 체험 모드에서도 JAWS에서 제공하는 대부분의 기능을 사용할 수 있습니다.
```

### 한국어 TTS 설치

JAWS는 다양한 TTS 엔진을 사용할 수 있으며, 한국어 TTS를 다운받아 설정하면 한국어 음성을 출력합니다. `Downloads > Synthesizers`로 이동하여 해당 페이지(https://support.freedomscientific.com/Downloads/Synthesizers)에서 한국어 TTS를 다운로드합니다. TTS 엔진은 MSI 형식의 파일로 쉽게 설치할 수 있습니다.

<img
  width="500"
  src="https://evie-ooooori.github.io/accessibility-guide-book-for-component/img/jaws-korean-tts.png"
/>

```plaintext
💡 영문 버전의 죠스를 설치 후 TTS를 적용하는 방법을 통해 JAWS를 최신 버전으로 유지하면서도 한국어 음성을 이용할 수 있습니다. JAWS 한글 버전은 실로암 시각장애인 복지관에서 한국어화하여 판매하며, 해당 기관에서 운영하는 아이프리 서비스에 등록이 되어 있어야만 사용할 수 있습니다. 또한, 최신 버전의 JAWS가 한국어로 번역되는 데에는 시간이 소요될 수 있기 때문에 최신의 기능과 개선 사항을 바로 이용하기 어려울 수 있습니다. 따라서 한국어 TTS를 사용하여 영문 버전의 JAWS에 적용하는 방법을 사용하는 것을 추천합니다.
```

### JAWS 실행하기

바탕화면에 바로가기 아이콘이 생성되었다면 해당 아이콘을 더블 클릭하여 실행 가능합니다.

```plaintext
💡 JAWS 2024 사용자 가이드 기준으로 설명합니다. 더 자세한 설명은 가이드 문서(https://support.freedomscientific.com/products/blindness/jawsdocumentation)의 내용을 통해 확인할 수 있습니다.

[추가] JAWS 사용법 교육: https://www.freedomscientific.com/training/jaws/
```

### 음성 설정

JAWS는 사용자 정의 가능한 음성 프로필을 지원합니다. 사용자는 다양한 음성 엔진, 언어, 목소리 등을 선택하고, 음성 출력의 속도, 피치, 볼륨, 구두점 처리 등을 조절한 후 설정을 프로필로 저장할 수 있습니다. 저장한 프로필은 나중에 불러와서 사용이 가능하며, 사용자의 필요에 따라 다양한 프로필을 생성하여 적절한 음성 설정을 간편하게 적용할 수 있습니다. 단축키 `Ctrl + JAWS + S`를 사용해 프로필을 변경할 수 있습니다.

`Options > Voices > Voice Adjustment` 로 이동 후 Voice Adjustment 대화 상자가 나타나면 필요에 따라 세부적인 음성을 설정합니다. 원하는 설정을 마치면 `Save As` 버튼을 클릭하여 새로운 프로필로 저장합니다.

### 알아두면 편리한 기능

#### 찾기

`Ctrl + F` 를 누르면 현재 페이지에서 찾고 싶은 내용을 쉽고 빠르게 찾을 수 있습니다. 입력창에 찾고자 하는 텍스트를 입력하면 검색 결과를 따라 초점이 이동합니다. 찾기 기능 사용 후 `JAWS + F3`을 누르면 다음 검색 결과로 이동하며, `JAWS + Shift + F3`을 누르면 이전 검색 결과로 이동할 수 있습니다.

  width="500"
  src="https://evie-ooooori.github.io/accessibility-guide-book-for-component/img/jaws-find.png"

#### 텍스트 뷰어

JAWS는 점자 뷰어와 텍스트 뷰어를 제공하여 개발자가 현재 출력 중인 정보를 시각적으로 확인하고 검증하는데 매우 유용합니다. Utilities > Braille and Text Viewer > Show Text Viewer 를 선택하면 화면 상단에 텍스트 뷰어가 활성화 되어 현재 출력 중인 정보를 확인 가능합니다.
